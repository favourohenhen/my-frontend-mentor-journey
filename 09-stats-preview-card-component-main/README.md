### What I learned
- When to use mix-blend-mode and background-blend-mode to achieve certain visual effects.
mix-blend-mode is used when the content of an element needs to blend with the content of its parent or sibling elements. background-blend-mode is used when the background of an element needs to blend with its own background layers.
